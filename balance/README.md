# バランス解析ツール

`game.js` の**実データ・実計算式**をそのまま使って戦闘バランスを自動検証するツールです。
データと式を二重管理しないよう、`game.js` をDOMスタブ環境（`harness.js`）で読み込み、
本体と同じ `getAtk()` / `getDef()` / 属性計算 / 防具スキルを呼んで評価します。

## 使い方

```bash
node balance/sim.js
```

各モンスターを「その段階で持っていそうな装備」で評価し、次を出力します。

- **撃**: 1撃あたり平均ダメージ（物理＋属性＋会心込み）
- **HTK**: 撃破に必要なヒット数
- **被**: 被弾ダメージ（通常 / 怒り、防御軽減後）
- **耐**: 何発耐えられるか（最大HP ÷ 被弾）
- **危**: 危険度（想定被ダメ ÷ 最大HP。100%超で要注意）
- 武器属性との相性（弱点! / 通常 / 不利）

`⚠簡単` `⚠硬い` `⚠危険` `ぬるい` を自動でフラグします。

## 前提パラメータ

`sim.js` 冒頭で、プレイヤーの挙動を仮定しています（タップ間隔・回避成功率・弱点会心率）。
ここを変えると評価の前提が変わります。

## 構成

| ファイル | 役割 |
|----------|------|
| `harness.js` | game.js をブラウザ無しで読み込み、内部データ・関数を取り出す |
| `sim.js` | 取り出した実関数で各モンスターを評価し、表で出力 |
