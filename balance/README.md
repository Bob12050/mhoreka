# バランス解析ツール

`src/` の**実データ・実計算式**をそのまま使って戦闘バランスを自動検証するツールです。
データと式を二重管理しないよう、本体と同じモジュール（`src/data/*`, `src/systems/equipment.ts`）を
直接インポートし、`getAtk()` / `getDef()` / 属性計算 / 防具スキルを呼んで評価します。

## 使い方

```bash
npm run balance
```

各モンスターを「その段階で持っていそうな装備」で評価し、次を出力します。

- **撃**: 1撃あたり平均ダメージ（物理＋属性＋会心込み）
- **HTK**: 撃破に必要なヒット数
- **被**: 被弾ダメージ（通常 / 怒り、防御軽減後）
- **耐**: 何発耐えられるか（最大HP ÷ 被弾）
- **危**: 危険度（想定被ダメ ÷ 最大HP。100%超で要注意）
- 武器属性との相性（弱点! / 通常 / 不利）

`⚠簡単` `⚠硬い` `⚠危険` `ぬるい` を自動でフラグします。

## 前提パラメータ

`sim.js` 冒頭で、プレイヤーの挙動を仮定しています（タップ間隔・回避成功率・弱点会心率）。
ここを変えると評価の前提が変わります。

|----------|------|
| `sim.ts` | `src/` の実データ・実関数を直接インポートし、各モンスターを評価して表で出力 |
