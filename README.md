# モホレカ (Mhoreka)

スマホで手軽に遊べる **モンハンNow風の位置ゲーム** の MVP です。
実際のGPSは使わず、**ゲーム内マップをタップで歩いてモンスターと遭遇 → 戦闘**する作りになっています。
Webアプリ（PWA）なのでアプリストア不要、URLを開くだけで遊べます。

## 遊び方

1. マップ上をタップすると、その方向にハンター（🧭）が移動します。
2. モンスターに近づくと戦闘が始まります。
3. **攻撃**でダメージを与え、相手の予告（⚠️）が出たら**回避**でかわします。
4. 討伐すると素材（🪨）がドロップ。討伐数（🏆）も記録されます。
5. 素材で**武器強化**して、より強いモンスター（🐉 古龍など）に挑戦！

### 亜種（強化変異個体）

ゲームを進めると、マップにまれに **亜種**（✨金色のリング）が出現します。

- 通常個体より **HP・攻撃が高く**、**弱点属性が変化**（別の属性武器が要る）
- 倒すと**レア素材「竜の宝玉」**をドロップ
- 宝玉で **最上位の武器・防具（覇玉シリーズ）** を生産できる
- 専用の**亜種クエスト**もあり

### クエスト

左下の **📜クエスト** ボタンで**クエストボード**を開けます。

- 「○○を△頭討伐せよ」という**狩猟クエスト**を受注（同時に1件）
- 対象を討伐すると進行（受注中は画面上部にバナー表示）、達成で**報酬**を獲得
- 報酬は素材のほか、節目のクエストでは**最大HPアップ**（初回のみ）
- エリア解放に応じてクエストも増え、最後は **古龍討伐** が最終目標

### 武器・防具（生産＆装備）

下中央の **🛠️装備・生産** ボタンから、モンハン式の装備システムを開けます。

- モンスターを倒すと**固有素材**（火竜の鱗・古龍の角など）がドロップ
- **武器**（大剣・太刀・ハンマー・弓など）を素材で**生産**して装備 → ⚔️攻撃力アップ
- **防具**（頭・胴・腕・腰・脚の5部位）を生産して装備 → 🛡️防御力で被ダメージ軽減
- 強い装備ほど**奥のエリアの素材**が必要（エリア進行と連動）

#### 防具スキル

防具にはそれぞれ**スキルポイント**が付いており、装備の合計が一定値に達すると**スキルが発動**します（装備画面の上部に発動スキルを表示）。

| スキル | 効果 |
|--------|------|
| 攻撃 | 攻撃力アップ |
| 防御 | 防御力アップ |
| 体力増強 | 最大HPアップ |
| 会心 | 一定確率で会心（大ダメージ） |
| 回避性能 | ジャスト回避の受付が広がる |

部位を組み替えて**ビルド**を作れます（例: リオソウル装備は攻撃寄り、覇玉装備は会心・回避）。

### 属性システム

武器には **属性**（🔥火 / 💧水 / ⚡雷 / 🐲龍）を持つものがあり、モンスターには **弱点属性** があります。
**弱点属性の武器で攻撃すると属性ダメージが大きく上乗せ**されます（耐性属性だと激減）。

- 戦闘画面にモンスターの**弱点・耐性**を表示
- 図鑑にも討伐済みモンスターの弱点を記録
- 例: リオレウス（火竜）は💧水が弱点・🔥火に耐性 → 水属性武器が有効

> 弱点を突くには、その属性の武器を素材で生産しておくのがコツ。

### マップ（エリア）選択

画面上部の**エリア名バナー**をタップすると**マップ選択**が開きます。
エリアごとに出現モンスターと難易度が異なり、最初は「始まりの草原」だけ。
**討伐数が一定に達すると新しいエリアが解放**され、奥のエリアには**新種**がいます。

- 🌿 始まりの森（最初から / ドスジャギィ・ドスファンゴ・ドスランポス）
- 🌴 旧ジャングル（討伐9体で解放 / イャンクック・ゲリョス・イャンガルルガ）
- 🌋 火山帯（討伐20体で解放 / リオレイア・リオレウス・ディアブロス）
- ❄️ 龍ノ頂（討伐32体で解放 / クシャルダオラ・**古龍**）

> 古龍は最終エリアのラスボス枠。序盤の定番モンスターから順に強くなります。

### 討伐図鑑

右下の **📖図鑑** ボタンで、モンスターの**討伐記録**を確認できます。
未討伐は「？？？」のシルエット表示、討伐済みは**討伐数と強さ**を表示。
初めて倒した種は **✨新種発見！** として登録され、上部に**コンプ率（◯/◯種）**が出ます。

### 出現中モンスター一覧

左下の **📋出現中** ボタンで、近くにいるモンスターを**距離が近い順**に一覧表示します。
行を**タップするとそのモンスターまで自動で移動**します。

### 戦闘の駆け引き

- **🎯 弱点露出**… 戦闘中ときどき弱点が光ります。そのタイミングで攻撃すると**会心（大ダメージ）**。
- **⚡ ジャスト回避**… 攻撃の着弾直前ギリギリで回避すると**反撃チャンス**。次の攻撃が大ダメージになります（早すぎる回避は安全だが反撃なし）。
- **🔥 怒り状態**… 相手のHPが減ると怒って攻撃が激化。ただし弱点も出やすくなる山場です。
- モンスターごとに**攻撃の速さ・回避猶予が違う**（素早い 🦇 / 重い一撃の 🐉 など）。

進行（武器Lv・素材・討伐数・HP）は**自動セーブ**され、次に開いたときに続きから遊べます。
やり直したいときは、HUDの **🏆討伐数を長押し**するとセーブをリセットできます。

## ローカルで動かす

Service Worker（PWA）はローカルファイル直開きだと動かないため、
簡易サーバーで配信するのがおすすめです。

```bash
# Python があれば
python3 -m http.server 8000
# → ブラウザで http://localhost:8000 を開く
# スマホで遊ぶなら、PCと同じWi-Fiで http://<PCのIP>:8000
```

## 構成

| ファイル | 役割 |
|----------|------|
| `index.html` | 画面構造（マップ・HUD・戦闘UI） |
| `style.css` | スマホ向けスタイル |
| `game.js` | ゲームロジック（移動・遭遇・戦闘・強化） |
| `manifest.webmanifest` / `sw.js` / `icon.svg` | PWA（ホーム追加・オフライン対応） |
